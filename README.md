# Image Slider / Carousel

A responsive image slider built using HTML, CSS, and JavaScript.  
Users can navigate through images using previous/next buttons, dots navigation, and automatic sliding.

## Features
- Manual navigation (Prev / Next buttons)
- Auto-slide functionality
- Dot indicators with active state
- Smooth CSS transitions
- Fully responsive design

## Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

## How It Works
- Images are displayed using a flex container
- JavaScript updates transform: translateX() for slide animation
- Dots and buttons control the active slide index
